## About  The Script
A script to access Email using IMAP server by imaplib library in Python 3.7+ to delete,count emails in a folder of Gmail or any other Email supported


### Written in

* [Python](https://www.python.org)


## Getting Started

Simple to use fully  automated Script 

### Prerequisites

Python 3.7+

* Python 3.7+ or higher Version
  ```python3
  python --version
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/amsynist/gmail_counter_deleter.git
   ```
2. Choose the cloned directory and run
   ```sh
   cd "directory path"
   python gmail_counter_deleter.py
   ```

Follow Instructions after running the script 
Note : Only enter numbers 1,2 while selecting options
